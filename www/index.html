This is the web interface for Oakfoam. It consists of a small web server together with some HTML and JS code. It is designed to be a simplistic and easy to use interface.

Usage Instructions: Click on the board to place a stone or select one of the menu buttons on the left. If you are unsure what a button does, hovering over it should display an informative tooltip.

For learning the rules of Go, try the Interactive Way to Go.

For a more advanced interface, try GoGui.

For playing against another human, try KGS.
